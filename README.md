# Kent Hack Enough *Attendees*

This is what the attendees see when they go to the KHE website.

## Setup

### Development
1. Install [Vagrant]()
2. Install [VirtualBox]()
3. Clone this repository
4. `vagrant up`
5. `npm install`
6. `bower install`
7. `gulp watch`

### Production
1. Clone this repository
2. `./provision.sh`
3. `bower install`
4. `npm install`
5. `gulp build`
